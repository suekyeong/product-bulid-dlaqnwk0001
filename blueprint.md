# 로또 번호 생성기

## 개요

이것은 5개의 고유한 로또 번호를 생성하고 표시하는 간단한 웹 애플리케이션입니다. 숫자는 당구공과 유사한 방식으로 표시됩니다.

## 기능

*   1과 45 사이에서 6개의 고유한 난수를 생성합니다. (오름차순 정렬 제공)
*   사용자 요청에 따라 3개의 번호 세트를 생성하여 "당구공" 안에 표시합니다.
*   **각 열(1~6열)의 숫자가 순차적으로 나타나는 애니메이션 효과.**
*   **Formspree를 이용한 제휴 문의 양식.**
*   새로운 번호 세트를 생성하는 버튼.

## 디자인

*   **레이아웃:** 제목, 로또 공, 버튼이 있는 단순하고 중앙에 배치된 레이아웃.
*   **스타일:** 로또 공은 3D 효과를 주기 위해 색상과 그림자가 있는 둥근 스타일입니다.
*   **상호작용:** "번호 생성" 버튼은 번호 생성을 트리거하고 디스플레이를 업데이트합니다.

## 현재 계획

이것은 애플리케이션의 첫 번째 버전입니다. 기본 HTML 구조를 만들고, CSS로 요소의 스타일을 지정하고, 숫자 생성 및 표시를 처리하는 JavaScript 로직을 추가할 계획입니다.
